#  Node.js Application Deployment using Jenkins CI/CD

## 📌 Project Description

This project demonstrates how to automate the deployment of a Node.js application using Jenkins.

Whenever code is pushed to GitHub, Jenkins automatically:
- Pulls the latest code
- Installs dependencies
- Deploys the application to the target server

This eliminates manual deployment and ensures faster and reliable releases.

---

## Tech Stack

- Node.js
- Jenkins
- Git & GitHub
- Linux (Ubuntu Server)
- NPM

---

## CI/CD Architecture

Developer → GitHub → Jenkins → Target Server

1. Developer pushes code to GitHub
2. Jenkins triggers build
3. Code is checked out
4. Dependencies installed
5. Application deployed
